# ChatGPT 学生对话场景总结

> 整理自 ChatGPT 官方学生使用页面及相关研究资料

## 概述

ChatGPT 为学生用户提供了 **100个对话场景模板**，涵盖学术学习、职业发展、生活问题等多个方面。这些模板由学生创建并针对学生需求定制。

### 使用统计数据

| 学生群体 | 使用率 | 主要用途 |
|---------|--------|---------|
| 高中生 | 69% | 作业辅导 |
| 英国大学生 | 88% | 学术工作 |
| 全球高等教育学生 | 86% | 各类学术任务 |

## 五、三大核心使用场景

根据研究，学生使用 ChatGPT 近 80% 集中在以下三类：

| 使用场景 | 占比 | 具体内容 |
|---------|------|---------|
| 实用建议 | ~30% | 获取问题指导、决策支持 |
| 信息搜寻 | ~25% | 研究、事实查证、资料收集 |
| 写作辅助 | ~25% | 起草、编辑、改进内容 |

## 六、优质提示词模板结构

### 6.1 标准结构

```
1. 角色扮演：告诉 AI 应该扮演什么角色
2. 信息说明：说明用户会提供哪些信息，AI 应该如何处理
3. 具体指令：将第一条具体指令放在引号中
4. 输出格式：指定回答的格式和长度
```

### 6.2 示例对比

| 普通 Prompt | 优化后的 Prompt |
|------------|----------------|
| "帮我写简历" | "你是一位资深的人力资源专家。我将提供我的工作经历和技能，请帮我优化简历内容，使其更具吸引力。输出格式为分点列出，每点不超过 50 字。" |
| "解释量子力学" | "请用简单的类比和生活中的例子，向一个高中生解释量子力学的基本概念，重点说明波粒二象性和不确定性原理。" |
| "怎么学好英语" | "我是中级英语学习者，目标是提高口语能力。请制定一个 3 个月的学习计划，每周 5 小时，包含具体的学习资源和练习方法。" |

## 七、使用注意事项

| 注意事项 | 说明 |
|---------|------|
| 验证信息 | AI 可能产生错误信息，重要内容需要核实 |
| 学术诚信 | 了解学校关于 AI 使用的政策，合理使用 |
| 隐私保护 | 不要在对话中透露敏感个人信息 |
| 批判思考 | 对 AI 的回答保持批判性思考，不盲目接受 |
| 学习辅助 | 将 ChatGPT 作为学习辅助工具，而非替代学习 |
