# 9大关键AI概念全图解与硬核推导

这九个概念涵盖了从 LLM 输入处理到模型微调的核心技术栈，每个概念都从直观理解出发，深入到信息论、概率论和最优化层面的数学原理。

---

## 1. Tokenization（词元化）

词元化是将自然语言文本拆解为模型可理解的最小单位（Token），大小介于字符和完整单词之间。自然语言是一个近乎无限且不规则的表示空间，词元化的本质是在词表大小和序列长度之间找到最优平衡——这本质上是一种数据压缩算法。

目前最主流的算法是 **BPE（Byte-Pair Encoding）**，其核心是逐步合并高频相邻 Token 对：先将词汇库拆为单字符，统计相邻 Token 的共现频率，选出最高频的对合并为新 Token，重复迭代直到词表达到预设阈值。从信息论角度看，BPE 实际上是在寻找使语料库编码后信息熵最小的字典——通过合并高频字首字根，让组合后的概率变大，减少分布长尾效应，从而压缩自然语言的平均表示维度。

## 2. Text Decoding（文本解码）

大模型本质上是一个「单词猜谜机」，每次输出下一个单词的概率分布。文本解码的核心问题就是：拿到这些概率后，如何决定究竟「说」哪个词从而串成一段话。

自回归语言模型在当前步骤 $t$ 给出条件概率分布 $P(x_t | x_1, x_2, \dots, x_{t-1})$，解码策略主要有三种：**Greedy Search** 每次直接取局部概率最大的词，容易陷入局部极值；**Beam Search** 在贪婪与穷举间折中，每层保留 $K$ 个累积概率最大的路径（$K$ 即 Beam Size），路径打分转为对数域相加避免数值下溢；**Nucleus Sampling（Top-p）** 为增加生成随机性，将词典按概率降序排列，找到累积概率刚好超过阈值 $p$ 的最小集合，在该截断集合内按相对概率重新归一化后随机采样，既截断长尾低概率词避免乱码，又保持词汇多样性。

## 3. Prompt Engineering（提示工程）

提示工程是与 AI 对话的「咒术」——通过提供背景、规则约束或示例，就能让模型完成从未见过的新任务。一个只被训练用来「预测下一个词」的模型，为什么改给它看几张示例（Few-Shot）就能做情感分类、写代码？

前沿研究（如北大和清华的《Why Can GPT Learn In-Context?》）在数学上证明了：**In-Context Learning（ICL）很大程度上等同于进行了一次隐式梯度下降微调**。当在 Prompt 中给出 Few-Shot 样本集并附加要预测的内容时，在 Transformer 自注意力机制的作用下，前向传播生成的 Key 和 Value 的内积加权平均，其数学形式同构于用这些样例对「模型权重的一层」进行梯度下降更新。换句话说，优秀的 Prompt 等同于在推理阶段向模型注入了高质量的梯度更新方向，将通用模型「校正」为专用模型。而 **Chain of Thought（CoT）** 则通过将映射拆分为多步，让每一步的熵降大幅变小，降低函数逼近难度。

## 4. Multi Step AI Agents（多步AI智能体）

让 AI 拥有「手和脚」——不仅能进行文本回答，还能通过明确的「思考」调用外部工具（浏览器、代码解释器等），持续观察结果，自动分步骤解决一个大问题。

Agent 在环境中的交互在强化学习中常被称为**部分可观测马尔可夫决策过程（POMDP）**。以最主流的基于 LLM 的智能体架构 **ReAct（Reasoning and Acting）** 为例：传统强化学习仅仅依赖策略 $\pi(a_t | s_t)$，而在大模型 Agent 中，基于语言的历史轨迹定义为 $\tau_t = (o_1, a_1, \dots, o_t)$，输出策略分成两轨并行，引入显式的思考痕迹 $z_t$。Agent 所谓「多步」，实际上是把不可微的外部世界环境反馈通过自然语言无缝注入回马尔可夫链条的上下文窗口中，使得 LLM 能够根据最新世界状态进行下一轮动作的重新动态规划。

---

## 5. RAGs（检索增强生成）

大模型的「开卷考试」系统。由于大模型可能包含过时信息或患有「幻觉」，RAG 会在它回答问题前，先去一个可靠文档库里搜索相关文本段落，然后把段落内容附在问题后面，让 AI 基于可靠证据给出结论。

RAG 将生成过程从纯参数化概率分布转变为结合非参数化外部数据库检索的联合概率分布。核心流程包括：**密集向量检索（DPR）** 使用双塔模型对查询和语料进行降维嵌入，相关度定义为度量空间里的内积；**ANN（近似最近邻搜索）** 通过 HNSW 或 IVFPQ 算法将穷举查找的复杂度从 $O(N)$ 降低到 $O(\log N)$；检索出的高质量文档极大压缩了生成器的分布方差，使模型概率迅速集中在事实相关的 Token 分布上。

---

## 6. RLHF（人类反馈强化学习）

大模型如果不加约束，很容易成为满口脏话、胡说八道或包含危险信息的机器人。RLHF 就是找一堆人类，像培训导盲犬一样，对大模型的多个回答选出「最好最安全的」，用这类数据进行强化学习，迫使 AI 模型向人类偏好价值观对齐。

对齐技术核心三步：SFT（监督微调）→训练 RM（奖励模型）→PPO（强化学习算子优化）。建立奖励模型时，利用人类标记者比对出的偏好对，基于 Bradley-Terry 模型将人类「主观胜负偏好」建模转化为可训练的潜在奖励得分差。PPO 强化优化时引入 KL 散度约束作为「弹簧」，一方面允许新模型去迎合更高的人类得分，同时拽住它不允许过度偏离基础分布。

---

## 7. VAE（变分自编码器）

图像和声音生成的先驱。相当于一个极强的「画手」，它先看海量画作，在大脑中把所有内容压缩成一组控制旋钮。作画时随机拨动旋钮，就能变出一张从未被画过的全新且真实的画。

VAE 是变分推断理论在深度学习中应用的典范。假设数据集每个样本背后都有隐变量生成，我们要最大化数据集整体对数似然。定义一个推断网络去逼近隐变量后验分布，通过数学拆解，要最大化的目标实际上成了证据下界（ELBO）：前项是重建损失，后项是先验正则化。**重参数化技巧** 将方差采样过程变换为 $z = \mu + \sigma \odot \epsilon$，使得噪声单独来自随机分布，网络直接对决定形状的 $\mu, \sigma$ 求导偏置。

## 8. Diffusion Models（扩散模型）

Midjourney、Stable Diffusion 与 Sora 底层的灵魂技术。类比：将一滴墨水滴在纸上渐渐摊开变成纯灰色，AI 要做的就是观察那最后一片灰色，逆向一步步收缩，将它复原成完美山水画。

它基于非平衡热力学的随机马尔可夫链原理。**前向过程**在 $T$ 个时间步内不断对信号施加高斯白噪声，由于高斯分布加法的可叠加性质，能得到从 $x_0$ 到 $x_t$ 的闭式解析解。**逆向过程**中，模型拟合神经网络来模拟后验概率倒退——关键发现是神经网络只需预测当前添加的噪声分布，不必预测上一步长什么样。训练损失函数异常简洁：最小化真实噪声和模型预测噪声的欧拉平方空间差值即可完成强大的图像生成收敛。

---

## 9. LoRA（低秩微调）

当你下载了一个百亿参数的 AI 想在自家电脑上教它新东西时发现显卡爆炸了，LoRA 就相当于只花费 1/1000 的显卡资源，在巨脑外部插上几根可更新权重的「信号小插排」，插排配上主脑就能输出堪比完全重置主脑的效果。

微软研究院发现一个自然规律：极大型预训练模型在微调解决特定任务时的「本征维度」极其微小。因此可以将巨大权重更新矩阵用低秩近似拆散为两个维数极小的矩阵联乘 $\Delta W = B \times A$，其中 $r$ 通常很小（典型取值 $r=8$ 甚至 4）。前向推断时将主参数大矩阵冻结，梯度只反馈在小矩阵上。参数更新量被缩减到不足百分之一，推断时合并为 $W_{\text{new}} = W_0 + BA$，不会引发计算延迟增加。
