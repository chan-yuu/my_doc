Isaac Sim 有两种典型的安装方式，这里优先选择预编译版本，比较稳定。

官方安装文档：https://docs.isaacsim.omniverse.nvidia.com/5.1.0/installation/quick-install.html

解压时用 unzip 还有点慢：

```bash
unzip isaac-sim-standalone-5.1.0-linux-x86_64.zip -d isaacsim
```

第二种是源码编译形式，开源了部分内容但大部分是下载下来的，并非完全开源方案。主要从中理解软件底层：

```bash
git clone https://github.com/isaac-sim/IsaacSim.git isaacsim
# 当前 main 使用 v5.1.0，v6.0 在 develop 分支

cd isaacsim
git lfs install
git lfs pull

./build.sh
# 期间可能会失败，需要重复运行直到显示 success
```

安装完成后通过 `isaac-sim.sh` 启动，可能出现错误需要手动指定环境变量，在 sh 中添加：

```bash
export LD_LIBRARY_PATH=/home/cyun/Project/IsaacSim/_build/linux-x86_64/release/exts/omni.isaac.ml_archive/pip_prebundle/nvidia/nvjitlink/lib:$LD_LIBRARY_PATH
```

能够顺利启动即可。

## 真正的安装形式——IsaacLab

现在的 Isaac Sim 已经几乎不单独安装了，而是直接与 IsaacLab 联动，在 pip 时就能安装 IsaacSim。想要使用模仿与强化学习的内容，需要完整安装 IsaacLab + IsaacSim。

官方文档：https://isaac-sim.github.io/IsaacLab/v2.1.0/source/setup/installation/index.html

以后的安装方式固定如下：

```bash
isaacsim
# 进入 isaaclab
./isaaclab.sh -p
```

IsaacLab 可以直接创建环境，如果需要对环境进行组合：

```bash
python /home/cyun/Project/isaac/IsaacLab/scripts/teleop/teleop_keyboard.py --task Template-Test-Direct-v0
```

这是一个远控脚本用于测试环境。
