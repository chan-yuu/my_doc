目前原生的 mac 有很多地方和 Windows 的体验可以说是相差甚远的，所以我们需要通过一些免费、开源的工具来提升使用体验。一下就是精选的各类软件。
首先是文件资源的管理，mac 的访达还是有点一言难尽的，我使用的是 double commander 来提升使用的体验，安装后需要输入：
```bash
xattr -cr "/Applications/Double Commander.app/"
```
即可正常使用。

第二个是mounty，这个是用来连接 microsoft 的 ntfs 文件格式和 mac 的，也就是让 mac 能够读写外接的硬盘的最佳工具。

第三个是 raycast，这是一个偏工具箱的东西，并且是一个可以使用快捷启动命令的工具，也就是 Windows 里面的一部分的工具。这里我们就主要录制的是 wezterm 的快捷启动了。
